# CommandDeck Roadmap

Blueprint order (do not reorder without explicit approval):

1. Secure approved-command runner
2. Runtime/machine selector
3. Live stdout/stderr streaming
4. Saved command history and evidence logs
5. Preview registry
6. Runtime health board
7. Docker/devcontainer workflow support
8. GitHub workflow bridge (later)
9. Full terminal access (later)

## Notes

- CommandDeck is currently in pre-runner state: login/auth branding and local workspace boot are validated.
- Priority remains safety, auditability, and deterministic behavior before broader execution capabilities.
